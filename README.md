# Project Overview
EngiCalc PH is a powerful engineering calculator designed to provide engineers and students with the tools necessary for efficient calculations and analysis.

## Goals
1. To offer a comprehensive suite of engineering formulas and tools.
2. Enable offline functionality for ease of access.
3. Incorporate AI features for predictive and intelligent calculations.
4. Support local-only distribution for privacy and data security.

# Offline/AI Pack Concept
The offline/AI pack aims to provide users with essential tools that can function without an internet connection, while also leveraging AI capabilities for advanced computation and analysis.

# Local-only Distribution
This approach ensures that all user data remains private and secure, allowing users to work without the concerns of data leakage or online dependencies.
